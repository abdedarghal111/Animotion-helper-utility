Acabo de crear una herramienta en html, js y sass para cubrir la tarea de crear diapositivas de código para automatizar tareas en "Animotion" (FrameWork para crear presentaciones).

Este programa es una especie de editor de texto con páginas para facilitar la edición y visualización.

Funciones posibles:
- Clonar páginas para ambas direcciones.
- Navegar entre páginas
- Borrar páginas
- Importar presentaciones
- Exportar presentaciones
- Editar páginas
- Terminal que informa de mensajes relevantes

La interfaz es fea, pero lo que me importaba era finalizarlo lo antes posible y que cumpla con la función.

Importante, si vas a usarlo debes abrir un servidor que te provee la página. En VScode existen extensiones como "Live Server" que te realizan eso.
